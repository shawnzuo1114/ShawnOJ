# 🧭 ShawnOJ Backend

一个用于在线评测系统的后端服务，实现用户管理、题目管理、代码提交与评测等核心能力。本项目为个人练手项目。

------

## 📌 项目简介

本项目聚焦 Online Judge 系统的核心后端功能，包括用户体系、题目管理、提交记录以及后续可扩展的评测模块。
