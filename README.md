# 🧭 ShawnOJ Backend

本项目为在线评测系统（Online Judge, OJ）的后端服务，提供用户管理、题目管理、测试用例处理、代码评测流程、比赛管理等核心能力。系统采用模块化架构设计，适用于学习实践与进一步扩展。

---

## 📌 项目简介

ShawnOJ Backend 致力于构建一个稳定、清晰且可扩展的 OJ 系统后端。系统提供用户体系、题目与测试用例管理、比赛内容组织、评测调度等基础能力，同时支持后续增强，包括统计分析、缓存优化及安全策略。

整体 API 设计遵循 RESTful 规范，可适配前后端分离架构及多端访问场景。
